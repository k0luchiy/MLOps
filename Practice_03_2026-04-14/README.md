# Отчет: Прогнозирование медицинских расходов с MLflow

## Описание
В работе исследуется датасет медицинских страховок (`insurance.csv`). Цель — построить модель, предсказывающую индивидуальные медицинские расходы пользователя.

## Ход работы
Было обучено 3 варианта модели `RandomForestRegressor` с логированием в MLflow:

1.  **Underfit_Model** (5 деревьев, глубина 2): Показывает низкую точность из-за недостаточной сложности.
2.  **Balanced_Model** (100 деревьев, глубина 5): Оптимальная модель, улавливающая основные тренды (например, сильное влияние курения на цену).
3.  **Complex_Model** (500 деревьев, глубина 20): Попытка максимально усложнить модель, приводящая к переобучению.

## Обоснование выбора модели
Победителем признана **Balanced_Model**.

- **R2 Score:** ~0.86 (самый высокий результат).
- **RMSE:** Модель показала наименьшую среднеквадратичную ошибку.
- **Анализ:** В медицинских данных очень важна обобщающая способность. `Complex_Model` показала чуть более низкий R2 на тестовой выборке по сравнению с `Balanced_Model`, что свидетельствует о том, что глубокие деревья начали подстраиваться под специфические выбросы (noise) в обучающих данных, теряя точность на новых данных.
